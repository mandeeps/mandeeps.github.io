Personal Projects

Cheer Up!

A web app that displays random but hopefully adorable photos from the imgur /r/aww gallery.

It can be run in supported browsers from this GitHub page and saved to the homescreen on iOS and Android.

Install on FirefoxOS from the Firefox Marketplace where it has a 5 star rating, over 5000 installs and was a featured app in the Photo and Video section.

The HTML5 version can be installed through the Amazon Appstore for Android. A PhoneGap packaged version for Android is also available on the Google Play Store.

Source is on GitHub. Inspired by Pocket Kitten. Written in 2013.

Barbell Calculator

A mobile web app to determine what plates to place on a barbell to lift a specified weight. Packaged with Cordova and available on the Google Play Store as an ad-supported free app using AdMob, and a paid ad-free version.

Also available in paid and free versions on the Amazon Appstore for Android.

Source is on GitHub. Built with ChocolateChipUI. Written in 2014.

TipCalc

A simple no hassle tip calculator for Android without ads or fluff. Also on the Amazon Appstore.

Source is on GitHub. Written in Java, in 2013.

Stellarnaut

A Flash game in the style of Space Invaders where you control a lone ship and battle through an alien fleet to reach and destroy the enemy mothership. 35000 total play sessions tracked by Mochi analytics.

Written in ActionScript 3 with Flashpunk. Developed in Linux without Adobe CS, using the Flex SDK. Source is on GitHub. Written in 2011.



Admiral Gobi

A Flash game where you fight waves of pirate ships, made for a relative with historical interests. 82000 total game sessions according to Mochi analytics.

Also built using Flashpunk and the Flex SDK on Linux. Source is on GitHub. Written in 2010.

Cat Links

A social link sharing site (like reddit) for cat fanatics supporting user accounts and commenting, built on Meteor. Created for the Coursera Introduction to Meteor course final project in December 2015. Source on GitHub.

Also available in paid and free versions on the Amazon Appstore for Android.

Create A Meme Android App

MEAN Rock Paper Scissors

A MEAN stack (MongoDB/Express/AngularJS/NodeJS) demo app witten in CoffeeScript and running on Heroku with MongoHQ, using Twitter Bootstrap for a mobile-first responsive layout. The GH-pages version connects to Heroku over a REST API and is packaged for Android through PhoneGap Build. Source is on GitHub. Written in 2013.

TI MSP430 Cylon LED Chaser

A simple project to learn about microcontrollers and electronics. Written in C for the Texas Instruments MSP430 Launchpad line of 16bit microcontrollers in 2010 using mspdebug and mspgcc4.

Here's a video showing a running perfboard version I made to practice soldering, with 1600 views. Source is on GitHub, simple as it is.
