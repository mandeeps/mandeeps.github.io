Projects

Cheer Up!

A web app that displays random but hopefully adorable photos from the imgur /r/aww gallery.

It can be run in supported browsers from this GitHub page and saved to the homescreen on iOS and Android.

Install as an open web app on FirefoxOS, Android and desktop versions of Firefox from the Firefox Marketplace. HTML5 version can be installed through the Amazon Appstore for Android. A PhoneGap packaged version for Android is also available on the Google Play Store.

Source is on GitHub. Inspired by Pocket Kitten. Written in 2013.

Stellarnaut

A Flash game in the style of Space Invaders where you control a lone ship and battle through an alien fleet to reach and destroy the enemy mothership. 32000 total play sessions tracked by Mochi analytics.

Distributed across the web by Mochi Media, built to use their APIs and written in ActionScript 3 with Flashpunk. Developed in Linux without Adobe CS, using the Flex SDK. Source on GitHub. Written in 2011.

Admiral Gobi

A Flash game where you fight waves of pirate ships, made for a relative with historical interests. 65000 total game sessions according to Mochi analytics.

Also built using Flashpunk and the Flex SDK on Linux, and distributed through Mochi. Source on GitHub. Written in 2010.

TI MSP430 Cylon LED Chaser

A simple project to learn about microcontrollers and electronics. Written in C for the Texas Instruments MSP430 Launchpad line of 16bit microcontrollers in 2010 using mspdebug and mspgcc4.

Here's a video showing a running perfboard version I made to practice soldering, with 1600 views. Source on GitHub, simple as it is.
